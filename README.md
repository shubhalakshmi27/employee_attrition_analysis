# HR Employee Attrition Analytics & Prediction Dashboard

An end-to-end Machine Learning and Business Intelligence solution designed to help HR departments understand historical turnover trends and proactively flag active employees at risk of resigning.

## 🚀 Key Features
- **Interactive HR Metrics:** Instant view of headcount, retention rates, and demographic breakdowns.
- **Predictive Analytics:** Implements a Random Forest Classifier trained on employee attributes (income, age, overtime) to evaluate resignation risk.
- **Actionable Risk Explorer:** An interactive dashboard tool allowing HR professionals to drill down into high-risk groups by department.

## 🛠️ Tech Stack
- **Language:** Python 3
- **Machine Learning:** Scikit-Learn (Random Forest Classifier)
- **Data Visualizations:** Plotly Express
- **Web App Dashboard Framework:** Streamlit
